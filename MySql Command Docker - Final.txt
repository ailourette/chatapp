Successfully got it to run mysql on docker container.

Working: https://www.youtube.com/watch?v=X8W5Xq9e2Os
1.) Download Docker Image
docker pull mysql/mysql-server:latest

2.) Run docker container
docker run -p 3307:3306 --name mysql -e MYSQL_ROOT_PASSWORD=password -d mysql

4.) Execute container
docker exec -it mysql /bin/bash

5.) Login to mysql
mysql -uroot -p -A
enter password:password

6.) To create a table in sql database in container
 	a.) USE mysql;
	b.) Press enter
 	c.) CREATE TABLE Users (UserName VARCHAR(30) NOT NULL PRIMARY KEY, Password VARCHAR(256), FirstName VARCHAR(30), LastName VARCHAR(30), Language VARCHAR(30));
	d.) Press enter
